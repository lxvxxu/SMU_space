# <공강이세요?>
## 📖프로젝트 소개 (Introduction to the Project)
![공강이세요? 앱 화면01](https://github.com/lxvxxu/SMU_space/blob/main/01.png)
![공강이세요? 앱 화면02](https://github.com/lxvxxu/SMU_space/blob/main/02.png)


‘공강이세요?’는 현 시간에 어느 건물에 어느 강의실이 비어 있는지 알 수 있게 해주는 앱입니다. 사용자의 핸드폰의 시간에 맞출 수도 있고, 사용자 본인이 직접 시간과 요일을 설정해서 그 순간에 어느 강의실이 비어 있는지 확인 또한 가능합니다. 학교 내부의 약도를 앱 안에서 볼 수 있게 첨부함으로써 건물들이 어느 위치에 있는지, 현재 자신의 위치와 가장 가까운 건물을 사용자 스스로가 알아볼 수 있습니다.

이 앱을 통해서 상명대학교 학생들이 필요한 시간에 필요한 곳에서 공부를 할 수 있게 될 것이라 기대되며, 지금은 상명대학교의 강의실만을 대상으로 만들었지만 추후에 다른 학교의 버전으로도 개발해볼 수 있을 것입니다. 학교의 강의실 뿐만 아니라 숙박시설 예약 앱 등에도 확장 적용할 수 있을 것이라고 기대됩니다.
